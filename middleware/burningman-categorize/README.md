# BurningMan Categorize middleware

## How to use it

1. Define Burningman API endpoint as you APItools service URL `http://playaevents.burningman.com/api/0.2/{year}/`
2. Add more categories depending on criterias
3. Make call to http://YOUROWN.apitools.com/event

## Existing categories (BM 2014)

* *cere* for Ceremonies
* *prty* for Gathering / Party
* *work* for Class / Workshop
* *care* for Care / Support
* *kid* for Kid-friendly
* *adlt* for Adult-oriented
* *game* for Game
* *para* for Parades
* *fire* for Fire
* *food* for Food
* *perf* for Performance
* *none* for Uncategorized
